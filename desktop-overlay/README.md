# HeadyE - AI Desktop Companion

## Core Features
- Desktop overlay UI
- Integrated task assistant
- Canvas desktop integration
- Comet browsing experience

## Stack
- **Frontend**: Electron + React
- **AI Core**: Python + Node.js
- **Canvas**: Custom WebGL renderer
- **Browser**: WebView2 + Extensions
